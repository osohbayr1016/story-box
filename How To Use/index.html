Introduction

  • Hello! This documentation is designed to help the admin understand the functionalities of the admin panel more effectively.

Dashboard

  • This is the dashboard. The admin can select a date range to view data for a specific period, as shown in the image below.
  • The dashboard displays various functionalities related to videos, posts, users, and songs with date range filter data.

User

  • This page is part of the StoryBox Admin Panel, designed for managing user accounts. It allows the admin to view, search, block/unblock, and perform actions on user accounts.
  • The search bar allows the admin to find specific users by entering relevant keywords such as user ID, email, username, etc.

Film Category

  • This page provides the admin with comprehensive tools to manage Film Category effectively. It offers functionalities for toggle active status, and create new categories within the StoryBox platform.
  • It offers to create a new film category within the StoryBox platform.

Film list

  • The "Film List" page allows the admin to view, active, inactive, delete, and perform actions on the films uploaded on the StoryBox platform.
  • This page allows the admin to create a new film to the StoryBox platform.

Episode List

  • The "Episode List" page allows the admin to view, delete, and perform actions on the episodes uploaded on the StoryBox platform.

Coin Plan

  • The "Coin Plan" page allows the admin to add, view, modify, toggle active status, and perform actions on the coin plans.
  • The admin can create and manage a new coin plan within the StoryBox platform.

VIP Plan

  • The "Vip Plan" page allows the admin to add, view, modify, delete, and perform actions on the VIP plans.
  • Admin can create and manage a new VIP plan within the StoryBox platform.

Order History

  • The "Order History" page allows the admin to view the coin plan purchase history of the users.
  • The "Order History" page allows the admin to view the VIP plan purchase history of the users.

Reward

  • This page represents the "Ads Coin Reward" management section of the Story Box Admin Panel. Admins can configure rewards by specifying the maximum ads per day, adding or modifying reward intervals, and defining coin values.
  • The interface also provides options to edit or delete existing entries in the reward list.
  • This page showcases the "Daily Coin Reward" section in the Story Box Admin Panel.
  • Admins can manage rewards assigned for consecutive daily activities, specifying the reward amount, associated day, and creation date.
  • Actions like editing or deleting rewards are also available for each entry.
  • This page represents the "Referral Reward" section in the Story Box Admin Panel.
  • Admins can configure referral-based rewards by defining the number of members required and the coins earned per referral.
  • A submission option allows saving the reward configuration.
  • This page allows administrators to configure the login reward system by specifying the number of coins users will receive upon logging in.

Setting

  • Notification Setting: Configure Firebase push notifications by adding the Private Key JSON file.
  • Non-VIP Access Setting: Set the number of free episodes accessible to non-VIP users.
  • Episode Duration Setting: Specify the duration of short episodes in seconds.
  • App Settings: Add or update links for the Privacy Policy and Terms of Use. Click Submit to save any changes.
  • Razor Pay Setting: Enable or disable Razor Pay and configure the Razor Pay ID and Secret Key.
  • Stripe Pay Setting: Enable or disable Stripe and set the Publishable Key and Secret Key.
  • Google Play Setting: Enable or disable Google Play as a payment method.
  • Flutter Wave Setting:Enable or disable Flutter Wave and configure the Flutter Wave ID.
  • Android Settings:
    • Enter the Interstitial Ad ID for Android in Android Google Interstitial.
    • Enter the Native Ad ID for Android in Android Google Native.
  • iOS Settings:
    • Enter the Interstitial Ad ID for iOS in iOS Google Interstitial.
    • Enter the Native Ad ID for iOS in iOS Google Native.
  • This page displays and manages the platform's currency settings in a tabular format.
  • Administrators can view, add, edit, or delete currency entries, including details like the currency name, symbol, code, country code, activation status, and creation date.
  • Use the New button to add a new currency.

Profile

  • Here, the admin can manage their own profile, change their password, name, email, and avatar.

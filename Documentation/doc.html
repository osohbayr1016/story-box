How to get Branch io key and domain.
  • Create account on Branch io
  • After creating Branch io Account got to Dashboard
  • Click Create New App
  • Enter your App name and then Create your app
  • For Android
  1. Configuration Click.
  2. Make sure I have an Android App and Enable App Links are both enabled
  3. Enter your URI scheme seen in below the page
  4. Select Custom URL and com.example.test with your app package name path:- android app level build.gradle file
  5. SHA256 cert Fingerprints see this link
  6. Then save button
  • For iOs
  1. Configuration Click.
  2. Make sure I have an iOS app is enabled
  3. Enter your URI scheme seen in below the page
  4. iOS the package name is the bundle identifier in Info.plist
  5. Then save button
  • Apple App Prefix is your Team ID from your Apple Developer Account
  • Change my app domain
    • On the Configuration page, in the General tab, navigate to the Link Domain section.
    • Choose a link domain which will be used for all your links:
    • Use My Own Domain
    • If you choose to use your own domain, enter your domain. The link domain is the website that hosts your deep links, it is not a deep link.
    • Change My app.link Subdomain
    • Here is where you can see and copy the current subdomain for your link domain. If you want to change it, click Change my app.link Subdomain.
    • Enter in a new subdomain.
    • Click Get when you're done.
    • If you have already changed your subdomain once, you will not have the option to change it again. If you need to make changes to the subdomain contact our Support team .
  • How to get Branch key
    • On the Account Settings page, in the General tab navigate to the Branch key and Sercret section.
    • And Copy your Branch key
